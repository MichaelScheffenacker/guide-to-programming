# Core Concepts
In a computer program *assignment* (*variables*), *repetition* (*loops*) and *branching* (`if`/`else`) allow to implement any algorithm (*Turing completeness*). (Turing completeness can also be based on the *lambda calculus* which utilizes functions only.) Branching also requires logic. Additionally to these concepts programming languages generally allow abstraction.
